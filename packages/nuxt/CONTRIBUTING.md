# Contributing

This guide covers package-specific development for `@posthog/nuxt`.

For repository-wide setup, see the root [CONTRIBUTING.md](../../CONTRIBUTING.md).

## Developing this module

1. Navigate into the module directory.
2. Install dependencies using `pnpm i`.
3. Build the module using `pnpm build`.
4. Navigate into the playground directory.
5. Install dependencies using `npm i`.
6. Build the playground using `npm run build`.
7. Run the playground using `node .output/server/index.mjs`.
