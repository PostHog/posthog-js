Initial release of `posthog-react-native-plugin`, the successor to `posthog-react-native-session-replay`. Adds support for native error tracking (iOS and Android crash autocapture via the underlying PostHog mobile SDKs) on top of the existing session replay functionality. Will be consumed in future versions of `posthog-react-native`.
