# Run the example

```bash
# root folder
pnpm build
pnpm package
pnpm package:watch
# example folder
pnpm install
# you can also avoid lockfile changes
pnpm install --no-lockfile
cd ios
pod install
cd ..
pnpm start
```

Open Xcode 16.4

Press the play button in Xcode to run the app.

Or...

```bash
npx expo run:ios
npx expo run:android
```

If your RN SDK changes are not picked up:

```bash
# example folder

# repeat Run steps
```
