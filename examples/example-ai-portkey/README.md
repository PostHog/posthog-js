# Portkey + PostHog AI Examples

Track Portkey AI gateway calls with PostHog via the OpenAI-compatible API.

## Setup

```bash
pnpm install
cp .env.example .env
# Fill in your API keys in .env
```

## Examples

- **chat.ts** - Chat completions via Portkey AI gateway

## Run

```bash
source .env
npx tsx chat.ts
```
