# Dedalus Labs + PostHog AI Examples

Track Dedalus Labs API calls with PostHog via the OpenAI-compatible API.

## Setup

```bash
pnpm install
cp .env.example .env
# Fill in your API keys in .env
```

## Examples

- **chat.ts** - Chat completions via Dedalus Labs

## Run

```bash
source .env
npx tsx chat.ts
```
